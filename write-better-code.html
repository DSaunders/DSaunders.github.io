Write better code

A short coding tip to your inbox at the start of every week

Better code is made up of small improvements.

At the start of every week I send out a small tip for how to improve your codebase this week.

It could be a common refactor, a performance tip, or perhaps something to improve readbility.

It might be something new to you, or simply a reminder of things you already know.

If we all focus on one small thing each week, we can all make our codebases incrementally better!
